# Enterprise Workforce Analytics & Resource Optimization System

## Project Overview
Built an end-to-end workforce analytics solution using SQL and Power BI.

## Tools Used
- PostgreSQL
- Power BI
- SQL
- DAX
- GitHub

## Key Features
- Workforce utilization analysis
- Overtime hotspot tracking
- Attrition analytics
- Staffing gap analysis
- KPI dashboards

## SQL Concepts Used
- JOINS
- GROUP BY
- HAVING
- CTEs
- Window Functions
- ROW_NUMBER()
- LAG()

## Dashboard KPIs
- Total Employees
- Attrition Rate
- Avg Utilization %
- Total Overtime
- Active Projects

## Business Insights
- Identified overloaded departments
- Analyzed workforce availability
- Detected overtime-heavy teams
- Measured staffing shortages
